------------------ Installation Instructions ------------------
To install the Windows licenses:

1. Open the DataHub properties, 
2. Select the Licenses icon,

If your licenses are in a file on disk,
  3. Select 'Load License File...'
  4. Select the file in the file dialog and press OK.

If your license is in an email message,
  3. Select 'Enter a License...' 
  4. Copy the license code (including all : characters) from your email,
  5. Paste the license code into the entry field.

Important: Do not install the same DataHub license on more than one computer.  Multiple TCP Link licenses can be installed on a single computer, but the same license cannot be installed on different computers.
